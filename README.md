# Tell Tale  
A tool for memorizing GRE vocabulary using short stories generated by ChatGPT.  

## Installation
```bash
pip3 install -r requirements.txt
```

## Run
1. Open a terminal and run  
```bash
python3 app.py
```

2. Go to the website and start learning!  
```bash
http://127.0.0.1:5000
```
